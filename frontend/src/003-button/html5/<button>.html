Según su tipo

Tipo Código Uso
button <button type="button"> </button> Botón simple, podemos añadirle eventos.
submit <button type="submit"> </button>

Cuando es presionado envía el formulario al que pertenece.

Podemos añadirle otros atributos como formaction, formenctype, formmethod, formnovalidate o formtarget para controlar como será enviado.
reset <button type="reset"> </button>

Restaura todos los campos del formulario a sus valores iniciales.

Los botones también aceptan un atributo form para enlazarlos al formulario al que pertenecen mediante el atributo id de este.
