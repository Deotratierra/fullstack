Las clases button de bootstrap pueden ser usadas dentro de elementos <a>, <button> o <input>. Referencia completa. Código fuente: SCSS, JS.

Según su aspecto

Aspecto Código
<button class="btn"> Base </button>
<button class="btn btn-default"> Default </button>
<button class="btn btn-primary"> Primary </button>
<button class="btn btn-success"> Success </button>
<button class="btn btn-info"> Info </button>
<button class="btn btn-warning"> Warning </button>
<button class="btn btn-danger"> Danger </button>
<button class="btn btn-link"> Link </button>

Según su tamaño

Aspecto Código
<button class="btn btn-primary btn-lg"> Grande </button>
<button class="btn btn-primary btn-md"> Mediano </button>
<button class="btn btn-primary btn-sm"> Pequeño </button>
<button class="btn btn-primary btn-xs"> Extra pequeño </button>

Botón de bloque

<button class="btn btn-primary btn-block"> De bloque </button>

Activado/desactivado

<button class="btn btn-primary active"> Activado </button>
<button class="btn btn-primary disabled"> Desactivado </button>
