Mostrar y ocultar elementos

Pulsa aquí para ocultar este elemento

Para ocultar elementos usamos la función hide() y para mostrarlos la función show(). Ambas toman dos parámetros, velocidad y callback. Para el parámetro velocidad podemos agregar tanto un número en milisegundos, como una cadena de texto como "slow" (600 ms) ó "fast" (200 ms). También podemos pasar los argumentos como pares claves-valor de un objeto.
