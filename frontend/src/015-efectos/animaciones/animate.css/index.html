Para animar un elemento añade las clase animated y luego añade una clase de animación. En el link anterior puedes consultarlas todas.

Para animar en bucle añade la clase infinite.

Para animar un elemento al recibir un evento, le añadimos la clase con addClass (jQuery), luego llamamos a la clase one() que permite consultar si la animación ha terminado. Recibe como primer parámetro un tipo de animación y como segundo un callback, donde immplementamos removeClass() (ver código JS).
