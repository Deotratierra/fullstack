Sistema de rejillas con CSS

CSS ofrece un sistema de rejillas muy interesante. Se basa en, a partir de un contenedor, este lo dividimos en filas y columnas para posicionar sus contenedores hijos en la fila y columna que necesitemos. Simplemente debemos definir al contenedor padre el estilo display: grid; o display: inline-grid;.

Con las propiedades grid-row y grid-column definimos el número de filas y columnas en las cuales se dividirá el contenedor.

Con las propiedades grid-template-rows y grid-template-columns definimos el tamaño de cada fila y columna. Pueden establecerse en fracciones (fr), píxeles (px) o en porcentajes (%). Básicamente, con estas propiedades definimos dónde comienzan las filas y columnas.

Las propiedades grid-column-gap y grid-row-gap indican la separación entre filas y columnas, respectivamente.


Ejemplo básico

En el siguiente ejemplo creamos un contenedor padre con display: grid; y un alto de 300px.

Indicamos que cada columna tenga la mitad del ancho con grid-template-columns: 2fr 2fr; (aunque también podríamos haberlod dejado automático con grid-template-columns: auto auto;) y el tamaño de las filas con grid-template-rows: 3fr 2fr 1fr;.

También definimos las propiedades grid-row-gap: 5px; y grid-column-gap: 20px; para separaciones entre filas y columnas.
