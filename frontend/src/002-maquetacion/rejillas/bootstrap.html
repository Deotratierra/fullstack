Containers

Trabajando con divs, usamos la clase container para centrarlos con un ancho fijo y container-fluid para un ancho completo en la página.


Rejillas

Bootstrap utiliza un sistema de filas con 12 columnas cada una para configurar fácilmente la disposición de los elementos. Todas las columnas utilizan un margen de ancho por defecto de 30px (15px en cada lado de las columnas). La siguiente tabla muestra el comportamiento y la sintaxis más simple al trabajar con rejillas para cada dispositivo:
