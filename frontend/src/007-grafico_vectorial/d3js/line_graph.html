Javascript

var margin = {top: 20, right: 20, bottom: 30, left: 50}, width = 960 - margin.left - margin.right, height = 500 - margin.top - margin.bottom; var parseTime = d3.timeParse("%d-%b-%y"); var x = d3.scaleTime().range([0, width]); var y = d3.scaleLinear().range([height, 0]); var valueline = d3.line() .x(function(d) { return x(d.date); }) .y(function(d) { return y(d.close); }); var svg = d3.select("#ejemplo1").append("svg") .attr("width", width + margin.left + margin.right) .attr("height", height + margin.top + margin.bottom) .append("g") .attr("transform", "translate(" + margin.left + "," + margin.top + ")"); d3.csv("https://raw.githubusercontent.com/mondeja/fullstack/master/frontend/src/007-grafico_vectorial/d3js/assets/data/line.csv", function(error, data) { if (error) throw error; data.forEach(function(d) { d.date = parseTime(d.date); d.close = +d.close; }); x.domain(d3.extent(data, function(d) { return d.date; })); y.domain([0, d3.max(data, function(d) { return d.close; })]); svg.append("path") .data([data]) .attr("class", "line") .attr("d", valueline); svg.append("g") .attr("transform", "translate(0," + height + ")") .call(d3.axisBottom(x)); svg.append("g") .call(d3.axisLeft(y)); });

Explicación

  • El siguiente ejemplo está extraído de aquí.
  • Referencia de D3JS.

  • Establecemos las dimensiones y los márgenes del grafo.

  • Creamos un parser para formatear las fechas. Desde aquí podemos consultar todos los posibles.

  • Establecemos los rangos. Para el eje x utilizamos scaleTime() # , que es una variante del rango lineal, pero coercionando los valores a fechas. Para el eje y utilizamos scaleLinear() # , una escala lineal simple.

  • Construimos un generador de línea con la función line # . Cuando se genera la línea, los accesores x e y son invocados por cada elemento del array de entrada.


  • Creamos el contenedor svg teniendo en cuenta los márgenes, moviéndolo de su posición con el tributo trasform, translate(x, y).




  • Extraemos la información del archivo con la función de petición csv() # , añadiendo un callback para manejar el posible error.

  • Recorremos el array de datos y parseamos las fechas. Podemos ver que la función csv nos ha convertido el archivo a un objeto con los cabeceros como claves.


  • Establecemos los dominios de x e y con las funciones extent() y max. La función max devuelve la fecha máxima para el valor de x en ese momento.


  • Creamos un path de clase line, en el cual volcamos la información con la función data() y la línea en el atributo d.


  • Añadimos dos grupos, uno para el eje x y otro para el eje y. En el caso del eje x, le aplicamos el atributo transform, translate(0, height) para situarlo en la parte baja del gráfico.
  • La función call()invoca a un callback para toda la selección. Con axisBottom() # y axisLeft() # indicamos la posición de los valores de los ejes.
