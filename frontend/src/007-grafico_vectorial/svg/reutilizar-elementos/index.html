Reutilizar elementos

Podemos crear elementos y volver a usarlos en diferentes partes. Tan sólo debemos insertarlo dentro de una etiqueta defs, proveerle un identificador y usar la etiqueta use, donde le indicaremos el atributo xlink:href apuntando al identificador.

Los siguientes elementos pueden ser reutilizados con defs:

  • Cualquier elemento de forma (line, rect, circle...)
  • Grupos (g)
  • Símbolos (symbol)
