Ruta del ejemplo:

d="M50,50 Q175,200 300,50"

Curvas Bézier cuadráticas

Para usar curvas Bézier cuadráticas usamos el comando Q dentro de la ruta, (en minúscula para puntos relativos y en mayúsculas absolutos).

Las curvas Bézier cuadráticas sólo toman un punto de anclaje. Si comprendes el funcionamiento de las curvas de Bézier con dos puntos de anclaje no deberías tener problemas en manejar estas.
