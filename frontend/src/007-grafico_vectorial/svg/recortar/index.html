Para crear un recorte, dentro de un elemento defs, creamos un elemento clipPath identificado. Dentro de este creamos una figura que aplicaremos a otro elemento. El otro elemento sólo mostrará los puntos comprendidos en la ruta del clipPath.

Para aplicar el recorte a un elemento le indicamos el atributo clip-path="url(#id_del_recorte)". En el ejemplo a la izquierda he aplicado un recorte rectangular a un círculo y he animado el recorte para que se aprecie bien.
