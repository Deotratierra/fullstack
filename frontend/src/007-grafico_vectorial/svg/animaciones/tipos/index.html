Animaciones en SVG

Las animaciones se realizan manipulando los atributos de formas a través del tiempo. Para ello usamos uno o más de los 4 elementos de animación, los cuales se colocan dentro de la figura:

Etiqueta Uso
<set> Establece un atributo en el tiempo
<animate> Anima un atributo de una figura
<animateTransform> Animación de una figura mediante transform
<animateTransform> Crea movimientos en figuras a través de rutas
Antes existía el elemento <animateColor> pero ha quedado obsoleto.


set

Set no funciona como una animación a no ser que sea encadenado múltiples veces. Accedemos al atributo que queremos animar por medio del atributo attributeName. Con los atributos from y to indicamos el rango del atributo. Con begin indicamos cuánto debe pasar hasta que se aplica el cambio.

Referencia de set

animate

En este elemento también se usan los atributos aplicados a set. Con dur indicamos la duración. Con repeatCount definimos las veces que queremos que se repita (para hacer un loop inifnito le pasamos el valor "indefinite").

Referencia de animate

animateTransform

Mediante el atributo type indicamos el tipo de transformación que aplicamos en la animación. Los atributos from y to van a depender en cuánto a número de parámetro del tipo de transformación que hayamos escogido.

Referencia de animateTransform

animateMotion

Con este elemento podemos indicar un path para moverlo a través. En este ejemplo estamos utilizando q, una curva de Bézier cuadrática para mover el rectángulo de arriba a abajo.
