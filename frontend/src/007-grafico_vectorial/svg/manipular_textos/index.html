Manipulación de textos con SVG

Con las etiquetas tspan podemos crear diferentes líneas de texto dentro de una etiqueta text y manipular sus posiciones. Por ejemplo, podemos usar el atributo dy para posicionarlas verticalmente o dx para posicionarlas horizontalmente.

Si proveemos un parámetro al atributo, el posicionamiento del texto dentro de la etiqueta será absoluto.

Si proveemos múltiples parámetros, cada letra se posicionará relativamente a la anterior los píxeles que corresponde a su número.


Realmente tspan es como las etiquetas span usadas en texto HTML normal, por lo que podemos aplicarles estilos como en el ejemplo a la izquierda.



Para hacer que el texto siga una ruta, debemos crearla con las etiquetas anidadas defs -> path, para luego usar la etiqueta textPath dentro del texto y referenciarla al identificador de la ruta con el atributo xlink:href (ver código).

En el ejemplo de la izquierda usamos el comando a dentro del path, el cual hace referencia a un arco elíptico, y, al estar en minúscula, sus puntos marcan coordenadas relativas.
