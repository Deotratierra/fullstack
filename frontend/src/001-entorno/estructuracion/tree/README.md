### Comandos útiles

#### tree
La utilidad de línea de comandos `tree` proporciona la salida de un árbol de directorios dentro de la ruta pasada como parámetro.
- Instalar (linux): `sudo apt-get install tree`
- Ejecutar: `tree .`
