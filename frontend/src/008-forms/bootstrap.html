Los formularios en bootstrap siempre reciben algo de estilo con bootstrap. Todos los elementos input, textarea y select con la clase .form-control también reciben un ancho del 100%.

Bootstrap provee tres tipos de formulario: verticales, horizontales y en línea. Las reglas estándar para los tres tipos son:

  • Envolver las etiquetas label # y los inputs en un div con la clase form-group.
  • Añadir la clase .form-control a todos los elementos input, textarea y select.

Bootstrap soporta los siguientes tipos de controles de formularios: input, textarea, checkbox, radio y select.


Desde aquí puedes acceder a ejemplos para añadir inputs con iconos asociados o con control del estado.

Desde aquí puedes ver ejemplos para cambiar el tamaño de los inputs al estilo bootstrap.

Código

<form> <div class="form-group"> <label for="email">Email:</label> <input type="email" class="form-control" id="email"> <span class="help-block"> Esto es un poco de texto de ayuda. <span> </div> <div class="checkbox"> <label class="checkbox-inline"> <input type="checkbox"> Opción 1</label> <label class="checkbox-inline"> <input type="checkbox"> Opción 2</label> </div> <div class="radio"> <label><input type="radio">Opción 1</label> </div> <div class="radio disabled"> <label><input type="radio" disabled>Opción 2</label> </div> <button type="submit" class="btn btn-primary">Enviar</button> </form>
<form class="form-inline"> <div class="form-group"> <label class="sr-only" for="email">Email:</label> <input type="email" class="form-control" id="email"> </div> <button type="submit" class="btn btn-primary"> Enviar </button> </form>

Explicación


  • Cada agrupaciónlabel e input va en un div de clase form-group.

  • Para insertar textos de ayuda usamos una etiqueta span con la clase help-block.




  • Para añadir selecciones de tipo checkbox o radio, insertamos los inputs dentro de elementos label.

  • Si queremos que la selección sea en línea le agregamos a los label las clases checkbox-inline o radio-inline.









  • Para añadir formularios en línea usamos la clase form-inline dentro del formulario.

  • Si no incluyes las etiquetas label, algunos lectores de pantalla pueden tener problemas los formularios. Puedes esconderlas para todos los dispositivos excepto para los lectores de pantalla usando la clase sr-only (screen reader only).
