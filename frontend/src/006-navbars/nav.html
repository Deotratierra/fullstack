Explicación

  • Cada barra de navegación debe estar envuelta en una clase navbar: nav class="navbar".
    La otra clase en este ejemplo, navbar-inverse es un estilo de bootstrap para las barras de navegación. Exsite también navbar-default, que es gris claro. Para dejar la barra fijada arriba añadimos la clase navbar-fixed-top.
  • Las barras de navegación son fluidas, es decir, ocupan el ancho completo del contenedor en el cual se alojan. Usando contenedores podemos limitarlas fácilmente.
  • La clase navbar-header es donde se ubica el logo y el nombre del sitio de forma responsive. Además, dentro colocamos el botón que mostrará en los botones de la barra de navegación en dispositivos móviles y tablets. Lo creamos colocando 3 span class="icon-bar" cada uno encima de otro. La clase navbar-toggle es la que hace el trabajo de expandir/ocultar la barra de navegación. El atributo data-toggle es una función javascript de bootstrap que hace el colapso, desde aquí podemos consultar sus opciones. El atributo data-target indica en qué elemento/s se aplicará el colapso.

  • La clase navbar-collapse indica al div que este es colapsable y la clase collapse que debe empezar colapsado (si la quitamos en dispositivos móviles la barra aparecería desplegada).
  • Las clases navbar-left, navbar-center y navbar-right se usan para alinear los elementos a la izquierda, centro y derecha respectivamente. Las clases nav y navbar-nav indican que esas listas (ul) son parte de la barra de navegación, si las quitamos pierden su estilo.
  • Para agregar botones agregamos la clase navbar-btn a un botón de bootstrap.


  • Para agregar formularios agregamos a una etiqueta form la clase navbar-form. Podemos crear agrupaciones de inputs mediante las clases input-group e input-group-btn.

Código

<nav class="navbar navbar-inverse" role="navigation"> <div class="navbar-header"> <a class="navbar-brand" href="#">NombreDelSitio</a> <button type="button" class="navbar-toggle" data-toggle="collapse" data-target="#navbarId"> <span class="icon-bar"></span> <span class="icon-bar"></span> <span class="icon-bar"></span> </button> </div>
<div class="navbar-collapse collapse" id="navbarId"> <!-- navbar-collapse --> <ul class="nav navbar-nav navbar-left"> <li><a href="#">A la izquierda</a></li> </ul> <ul class="nav navbar-nav navbar-center"> <li><a href="#">Centrado 1</a></li> <li><a href="#">Centrado 2</a></li> <li><button class="btn btn-primary navbar-btn">Botón</button></li> </ul>
<form class="navbar-form navbar-right"> <div class="input-group"> <input type="text" class="form-control" placeholder="Buscar"> <div class="input-group-btn"> <button class="btn btn-success" type="submit"> <i class="glyphicon glyphicon-search"></i> </button> </div> </div> </form> <ul class="nav navbar-nav navbar-right"> <li><a href="#"><span class="glyphicon glyphicon-user"></span> Registro</a></li> <li><a href="#"><span class="glyphicon glyphicon-log-in"></span> Login</a></li> </ul> </div> <!-- End of navbar-collapse --> </nav>
