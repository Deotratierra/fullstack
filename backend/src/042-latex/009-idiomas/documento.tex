Al cargarlos se modifican órdenes y entornos como los siguientes:

enumerate  \roman     \section
itemize    \fnsymbol  \subsection
\%        \alph     \subsubsection
\Alph

% -----------------------------------------------------

    %    Modificación de órdenes y entornos   %

Todas las órdenes modificadas se pueden reconfigurar.

Por ejemplo, por defecto babel usa la palabra "Cuadro" debajo de las tablas.
Para que aparezca tabla podemos hacer:
\fi
\usepackage[spanish, es-tabla]{babel}  % al importar, o después:
\renewcommand{\tablename}{Tabla}
