## CMake
Es un sistema de metaconstrucción de programas multiplataforma.

______________________________

### Instalación
#### Unix
- Instalar: `sudo apt-get install cmake`
- Comprobar: `which cmake`
- Version: `cmake --version`

______________________________

### Ejemplo básico

#### Unix
```
cd basico
cmake .
make
./foo
```
